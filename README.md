# CGPA Calculator

A simple C++ program that calculates a student's GPA and overall CGPA based on grades and credit hours.

## Features

* Input multiple courses
* Enter grades and credit hours for each course
* Calculate total grade points
* Calculate GPA for a semester
* Calculate overall CGPA
* Display course details and final results

## Technologies Used

* C++
* Visual Studio Code
* Git & GitHub

## Concepts Used

* Variables and Data Types
* Loops
* Conditional Statements
* Functions
* Arrays
* User Input and Output

## How to Run

1. Compile the program:

```bash
g++ CGPA_calc.cpp -o CGPA_calc
```

2. Run the executable:

```bash
./CGPA_calc
```
